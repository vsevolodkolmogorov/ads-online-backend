# Ads Application

## Описание

Ads Application — это приложение для управления объявлениями. Оно предоставляет функциональность для создания,
редактирования, удаления и отображения объявлений, а также для управления пользователями и их комментариями. Это
веб-приложение построено с использованием Spring Boot и PostgreSQL, с дополнительной поддержкой безопасности и Swagger
для документации.

## Технологии

В проекте используются следующие технологии и библиотеки:

- **Spring Boot** — основной фреймворк для создания приложения
- **Spring Data JPA** — для работы с базой данных
- **PostgreSQL** — используется как основная база данных
- **Spring Security** — для аутентификации и авторизации пользователей
- **Swagger (OpenAPI)** — для автоматической генерации документации API
- **Liquibase** — для управления миграциями базы данных
- **MapStruct** — для маппинга объектов
- **Lombok** — для упрощения кода (генерация геттеров, сеттеров и т.д.)
- **JUnit 5** — для тестирования приложения

## Основные функции

- **Управление объявлениями:**
    - Создание, обновление и удаление объявлений
    - Просмотр списка всех объявлений или конкретных объявлений

- **Комментарии:**
    - Добавление, обновление и удаление комментариев к объявлениям

- **Регистрация и аутентификация пользователей:**
    - Регистрация нового пользователя с возможностью аутентификации
    - Обновление информации о пользователе, включая изображений профиля и пароль

- **Безопасность:**
    - Роли пользователей: только администраторам доступно удаление объявлений и пользователей.
    - Аутентификация и авторизация с использованием Spring Security.

## Как запустить проект

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/your-repository-url.git

2. Перейдите в директорию проекта:

   ```bash
     cd ads

3. Соберите проект:

   ```bash
     mvn clean install

4. Запустите приложение:

   ```bash
     mvn spring-boot:run


5. Настроить базу данных:

    ```bash
   spring.datasource.url=jdbc:postgresql://localhost:5432/ads_db
   spring.datasource.username=your-username
   spring.datasource.password=your-password

6. Запустить фронтенд с помощью установленного Docker, нужно открыть командную строку и выполнить следующую команду:

   ```bash
     docker run -p 3000:3000 --rm ghcr.io/dmitry-bizin/front-react-avito:v1.21

7. После этого приложение будет доступно по адресу:

   API: http://localhost:8080 \
   Swagger UI: http://localhost:8080/swagger-ui.html \
   Frontend: http://localhost:3000 \
